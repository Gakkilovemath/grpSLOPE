The R package [grpSLOPE](https://CRAN.R-project.org/package=grpSLOPE) was used to perform the simulation studies presented in (the submitted journal version of) [Brzyski, D., Gossmann, A., Su, W., & Bogdan, M. (2016). *Group SLOPE &mdash; adaptive selection of groups of predictors*](http://arxiv.org/abs/1610.04960). Here we show the resulting figures, and link to the R code that can be used to reproduce these simulation studies.

## Figure 2

* [:page_facing_up: Simulation code for Figure 2 a](https://github.com/dbrzyski/gSLOPE/blob/master/R/grpSLOPE/simulations/figure_2a.R)
* [:page_facing_up: Simulation code for Figures 2 b-e](https://github.com/dbrzyski/gSLOPE/blob/master/R/grpSLOPE/simulations/figure_2bcde.R)
* [:page_facing_up: Visualization code for Figures 2 a-e](https://github.com/dbrzyski/gSLOPE/blob/master/R/grpSLOPE/figures/figure_2.R)
